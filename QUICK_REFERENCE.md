- 11 conversation types (daily recap, app drama, productivity roast, etc.)
- Generated by OpenAI/Anthropic
- User ratings for improvement
- Triggered by usage patterns

## ğŸ“ Key Design Decisions

**Why PostgreSQL?** JSON fields, complex queries, production-ready

**Why Celery?** Reliable background processing, scheduling

**Why Django?** Rapid development, excellent ORM, strong admin

**Why REST over GraphQL?** Simpler for this use case, better tooling
